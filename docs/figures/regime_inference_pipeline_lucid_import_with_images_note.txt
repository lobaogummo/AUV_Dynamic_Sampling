This Lucid/draw.io import file embeds the real thumbnails used in the PNG-style regime inference diagram. The thumbnails are separate image objects generated from the validated Step00, Step05 and Step08 arrays. Dictionary atom thumbnails use the temperature-like half of the real Step05 `canonical_dictionary.npz` components; the second half is the valid-mask encoding. They are not pixel-editable inside Lucidchart, but they can be moved, resized, copied or replaced. All labels, section boxes, callout boxes and arrows are editable vector/text objects.
