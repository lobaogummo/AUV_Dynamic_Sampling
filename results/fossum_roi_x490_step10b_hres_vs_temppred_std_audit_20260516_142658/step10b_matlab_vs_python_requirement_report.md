# Step10B MATLAB vs Python Requirement Report

## Does HRes need MATLAB?
No. HRes for all 370 days already exists and was generated by Python. The Python output exactly matched Filipa HRes on validation dates.

## Does TEMPpred/STD need MATLAB?
The current official workflow uses MATLAB to prepare DSS inputs, call `DSS.C.64.exe`, read realizations, compute TEMPpred/STD, and write predModels. That part has not yet been fully ported to Python.

## Can MATLAB be replaced?
Probably yes, with Python + `DSS.C.64.exe`, but only after porting and validating the GSLIB/DSS preparation and readback logic against existing October predModels.

## Safe path
1. Do not regenerate HRes for C01/C06; use existing HRes/ROI arrays.
2. Port or execute only the `runSimulations.m` layer for TEMPpred/STD.
3. Validate Python+DSS or MATLAB+DSS against October before trusting C01/C06 outputs.
